## 四、J技能（激光锁定）详细

| 属性 | 数值 |
|------|------|
| 类型 | 持续激光束 |
| 伤害 | 2 HP/帧 |
| 视觉 | 黄色光束 + 白色核心 + 命中粒子 |
| 追踪 | 自动追踪锁定目标 |
| 切换目标 | 目标死亡后切换最近敌人 |
| 位移加成 | +2 px/帧（移速5→7） |

**激光伤害计算**：
- 60fps下，1秒 = 60帧
- 每秒激光伤害 = 2 × 60 = **120 HP/秒**
