## 问题列表

### 1. 散弹中间空了
- **描述**: 偶数子弹时，中间没有子弹，导致打不中目标
- **原因**: 子弹均分角度，中间为空
- **修复**: 让中间子弹始终朝前，或改变角度分配算法

### 2. 散弹黄色扇形效果
- **描述**: 激活散弹时显示黄色扇形指示
- **修复**: 移除该视觉效果

### 3. 蓄力弹进度条不同步
- **描述**: 进度条用 /60 计算，但实际间隔 Lv.1=120, Lv.3=80
- **修复**: 进度条使用 chargeInterval 动态计算

### 4. 护盾和蓄力弹颜色混淆
- **描述**: 都是绿色系，容易混淆
- **修复**: 护盾用蓝色，蓄力弹用黄色

### 5. 激光无法伤害 Boss
- **描述**: 激光连锁后对 Boss 无伤害
- **原因**: Boss 不在 laserHitTargets 链中
- **修复**: 确保 Boss 也能被激光伤害
