# 规格说明：波次技能系统重构

## 变更范围

本文档定义7个新局内技能的详细规格，每个技能可独立升级4次，所有技能可并行触发。

---

## 技能1：穿透弹 (Pierce)

### 描述
子弹可以穿透多个敌人后继续飞行

### 属性
| 属性 | 数值 |
|------|------|
| 技能ID | `pierce` |
| 最大等级 | 4 |
| 初始穿透数 | 1 |
| 满级穿透数 | 5 |
| 每级增量 | +1 |

### 实现细节
```
playerBullets.pierceCount = 1 + pierceLevel
```
- 当 pierceCount > 0 时，子弹击杀敌人后不消失
- 每穿透一个敌人，pierceCount 减1
- pierceCount = 0 时正常销毁

---

## 技能2：散弹 (Scatter)

### 描述
发射多路扇形分布的子弹

### 属性
| 属性 | 数值 |
|------|------|
| 技能ID | `scatter` |
| 最大等级 | 4 |
| 初始路数 | 3 |
| 满级路数 | 7 |
| 每级增量 | +1 |
| 扇形角度 | 45度 |

### 实现细节
```
bulletCount = 3 + scatterLevel
spreadAngle = 45度
发射角度 = 面向角度 ± spreadAngle/2，均匀分布
```
- 子弹伤害为基础伤害的70%
- 子弹速度不变

---

## 技能3：激光 (Laser Chain)

### 描述
释放激光时可以从一个目标连锁到多个目标

### 属性
| 属性 | 数值 |
|------|------|
| 技能ID | `laser` |
| 最大等级 | 4 |
| 初始连锁数 | 1 |
| 满级连锁数 | 5 |
| 每级增量 | +1 |
| 连锁距离 | 150像素 |

### 实现细节
```
laserChainCount = 1 + laserLevel
```
- 激光命中第一个目标后，在150像素范围内寻找下一个最近敌人
- 连锁伤害为基础激光伤害的50%
- 最多连锁 (laserChainCount - 1) 次

---

## 技能4：电磁波范围 (EM Wave)

### 描述
K技能释放电磁波，伤害玩家周围圆形范围内的所有敌人

### 属性
| 属性 | 数值 |
|------|------|
| 技能ID | `emWave` |
| 最大等级 | 4 |
| 初始范围 | 100像素 |
| 满级范围 | 全屏(约1000像素，对角线) |
| 每级增量 | 约225像素 |
| 基础伤害 | 100 |

### 范围计算
| 等级 | 范围半径 | 说明 |
|------|---------|------|
| 0 | 100px | 小圆 |
| 1 | 325px | 中圆 |
| 2 | 550px | 大圆 |
| 3 | 775px | 巨大圆 |
| 4 | 全屏 | 对角线距离 |

### 实现细节
```
emWaveRadius = 100 + emWaveLevel * 225
emWaveDamage = 100 * (1 + emWaveLevel * 0.5)
```
- 电磁波从玩家位置向外扩散
- 扩散速度：12像素/帧
- 扩散到最大半径后消失
- 范围内的所有敌人受到伤害
- 消耗能量200（满能量）

---

## 技能5：充能加速 (Energy Charge)

### 描述
对敌人造成伤害时获得额外的能量充能

### 属性
| 属性 | 数值 |
|------|------|
| 技能ID | `energyCharge` |
| 最大等级 | 4 |
| 初始充能 | 10 |
| 满级充能 | 30/次伤害 |
| 每级增量 | +5 |
| 触发条件 | 每造成10点伤害 |

### 实现细节
```
energyPerDamage = 10 + energyChargeLevel * 5
```
- 每造成10点伤害，获得 (energyPerDamage / 10) 点能量
- 满级时：每10点伤害 = 30点能量，击杀单怪(~30HP) ≈ 满充能

---

## 技能6：蓄力弹 (Charge Shot)

### 描述
充能后发射高伤害单发炮弹

### 属性
| 属性 | 数值 |
|------|------|
| 技能ID | `charge` |
| 最大等级 | 4 |
| 初始充能时间 | 60帧(1秒) |
| 满级充能时间 | 60帧(不变) |
| 伤害倍率 | 3倍 |
| 每级增量 | +0.3倍 |
| 满级倍率 | 4.2倍 |

### 实现细节
```
chargeDamageMultiplier = 3 + chargeLevel * 0.3
chargeTime = 60帧（固定）
chargeDamage = DAMAGE * chargeDamageMultiplier
```
- 按住射击键蓄力，松开发射
- 蓄力期间显示充能进度
- 满级4.2倍伤害 = 15 * 4.2 = 63伤害

---

## 技能7：光刀 (Melee Blade)

### 描述
在玩家周围形成旋转的防御性光刀

### 属性
| 属性 | 数值 |
|------|------|
| 技能ID | `blade` |
| 最大等级 | 4 |
| 初始角度 | 45度 (1/8圆周) |
| 满级角度 | 225度 (5/8圆周) |
| 每级增量 | +45度 |
| 旋转速度 | 0.08弧度/帧 |
| 伤害 | 基础伤害的50% |
| 防御 | 消灭接触的敌弹 |

### 范围计算
| 等级 | 角度 | 圆周比例 |
|------|------|---------|
| 0 | 45度 | 1/8 |
| 1 | 90度 | 1/4 |
| 2 | 135度 | 3/8 |
| 3 | 180度 | 1/2 |
| 4 | 225度 | 5/8 |

### 实现细节
```
bladeAngleSpan = 45 + bladeLevel * 45
bladeDamage = DAMAGE * 0.5 * (1 + bladeLevel * 0.25)
```
- 光刀在玩家周围旋转
- 角度范围内的敌人受伤、子弹被消灭
- 半径固定为100像素

## 并行触发规则

所有7个技能可以同时激活，互不影响：

| 技能 | 触发方式 | 持续时间 |
|------|---------|---------|
| 穿透弹 | 被动常驻 | 永久 |
| 散弹 | 被动常驻 | 永久 |
| 激光 | 按J键激活 | 按住期间 |
| 电磁波 | 满能量按K | 单次释放 |
| 充能加速 | 被动常驻 | 永久 |
| 蓄力弹 | 按住射击键 | 蓄力1秒后发射 |
| 光刀 | 被动常驻 | 永久 |

- `pierceBuff` - 移除（整合到技能系统）
- `chargeBuff` - 移除（整合到技能系统）

## 数值平衡

### 击杀充能目标（满级充能加速）
- 敌人平均HP：50
- 每10伤害 = 30能量
- 击杀1个敌人 ≈ 150能量 ≈ 75%能量条

### 技能组合强度评估
- 穿透+散弹+激光：远程输出强
- 充能加速+电磁波：频繁释放K技能
- 光刀：近身防护

---

## 测试场景

1. **穿透弹**：同时面对5个重叠敌人，一发子弹全部穿透
2. **散弹**：扇形7路弹幕覆盖
3. **激光**：3个聚集敌人，激光连锁击杀
4. **电磁波**：满能量释放，中等范围击杀
5. **充能加速**：频繁造成伤害测试充能
6. **蓄力弹**：充能1秒后发射高伤害弹
7. **光刀**：敌弹接触光刀被消灭

---

## 优先级

1. 穿透弹（最简单）
2. 散弹
3. 充能加速
4. 蓄力弹
5. 激光连锁
6. 光刀
7. 电磁波范围（最复杂）
