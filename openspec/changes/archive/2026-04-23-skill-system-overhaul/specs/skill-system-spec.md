# 技能系统重构规格说明书

**变更名称**: skill-system-overhaul  
**日期**: 2026-04-23

---

## 1. 技能叠加规则

### 1.1 统一叠加系统
- 所有7个技能均支持叠加
- 最大叠加等级：**3级**（从4调整为3）
- 每次波次结束3选1，选择后对应技能+1级
- 满级技能不再出现在3选1选项中
- 同一技能可重复出现在多个选项中

### 1.2 技能等级显示
- 波次完成界面显示所有已拥有技能的当前等级
- 格式：`技能图标 技能名 Lv.X/3`

## 2. 穿透弹 (Pierce)

### 2.1 效果定义
| 等级 | 穿透目标数 | 子弹行为 |
|------|-----------|---------|
| Lv.0 | 0 | 击中1个敌人后消失 |
| Lv.1 | 1 | 穿透1个敌人后继续飞行 |
| Lv.2 | 2 | 穿透2个敌人后消失 |
| Lv.3 | 3 | 穿透3个敌人后消失 |

### 2.2 场景
- GIVEN 玩家拥有穿透弹 Lv.2
- WHEN 子弹发射并击中第一个敌人
- THEN 子弹继续飞行并可命中第二个敌人
- THEN 子弹命中第二个敌人后继续飞行
- THEN 子弹命中第三个敌人后消失

## 3. 散弹 (Scatter)

### 3.1 效果定义
| 等级 | 总路数 | 说明 |
|------|-------|------|
| Lv.0 | 1 | 只有主子弹 |
| Lv.1 | 2 | 主子弹 + 1路散弹 |
| Lv.2 | 3 | 主子弹 + 2路散弹 |
| Lv.3 | 4 | 主子弹 + 3路散弹 |

### 3.2 场景
- GIVEN 玩家拥有散弹 Lv.2
- WHEN 玩家发射子弹
- THEN 同时发射3路子弹（主子弹居中，散弹对称分布）

---

## 4. 蓄力弹 (Charge)

### 4.1 效果定义
| 等级 | 效果 |
|------|------|
| Lv.0 | 无蓄力弹 |
| Lv.1 | 解锁充能炮弹，每隔固定时间自动发射1个 |
| Lv.2 | 充能炮弹伤害+50% |
| Lv.3 | 充能炮弹发射频率+50% |

### 4.2 行为规范
- 充能炮弹与普通子弹独立，可同时存在
- Lv.1解锁后自动生效，无需额外操作
- Lv.2/Lv.3为强化效果，必须先有Lv.1

### 4.3 场景
- GIVEN 玩家拥有蓄力弹 Lv.1
- WHEN 游戏中
- THEN 系统自动定期生成充能炮弹
- THEN 充能炮弹独立于玩家普通射击

---

## 5. 光刀 (Blade)

### 5.1 效果定义
| 等级 | 效果 |
|------|------|
| Lv.0 | 无光刀 |
| Lv.1 | 解锁旋转光刀，可销毁来袭子弹 |
| Lv.2 | 光刀覆盖角度在Lv.1基础上+45° |
| Lv.3 | 光刀覆盖角度在Lv.2基础上+90° |

### 5.2 行为规范
- Lv.1解锁后光刀自动激活
- 光刀持续旋转，围绕玩家
- 光刀接触敌弹时销毁该敌弹
- Lv.2/Lv.3仅增加覆盖角度

### 5.3 场景
- GIVEN 玩家拥有光刀 Lv.1
- WHEN 有敌弹接近玩家
- THEN 光刀接触敌弹并销毁
- THEN 玩家不受伤害

---

## 6. 激光连锁 (Laser Chain)

### 6.1 效果定义
| 等级 | 连锁目标数 | 说明 |
|------|-----------|------|
| Lv.0 | 0 | 无连锁 |
| Lv.1 | 2 | 激光可连锁2个额外目标 |
| Lv.2 | 3 | 激光可连锁3个额外目标 |
| Lv.3 | 4 | 激光可连锁4个额外目标 |

### 6.2 场景
- GIVEN 玩家拥有激光连锁 Lv.1
- WHEN 激活激光并锁定目标A
- THEN 激光伤害目标A
- THEN 激光连锁到最近的额外目标B
- THEN 激光连锁到最近的额外目标C
- THEN 连锁结束

## 7. 电磁波 (EM Wave)

### 7.1 效果定义
| 等级 | 波纹次数 | 余波伤害比例 |
|------|---------|-------------|
| Lv.0 | 1 | 100% |
| Lv.1 | 2 | 20% |
| Lv.2 | 3 | 20% |
| Lv.3 | 4 | 20% |

### 7.2 行为规范
- 满能量释放电磁波
- 第一次波纹伤害100%，后续每次20%
- 波纹之间有短暂延迟

### 7.3 场景
- GIVEN 玩家拥有电磁波 Lv.1，能量满
- WHEN 玩家释放电磁波
- THEN 第一次波纹扩散，伤害100%
- THEN 短暂延迟
- THEN 第二次波纹扩散，伤害20%

## 8. 充能加速 (Energy Charge)

### 8.1 效果定义
| 等级 | 每10伤害充能能量 |
|------|-----------------|
| Lv.0 | 10 |
| Lv.1 | 15 |
| Lv.2 | 20 |
| Lv.3 | 25 |

### 8.2 场景
- GIVEN 玩家拥有充能加速 Lv.2
- WHEN 玩家造成20点伤害
- THEN 玩家获得40能量

## 9. UI显示规格

### 9.1 波次完成界面
- 顶部：显示"第X波完成！"
- 技能状态栏：显示所有已获得技能的当前等级
  - 格式：`[图标] 技能名 Lv.X/3`
- 3选1卡片区域：显示3个可选技能

### 9.2 技能满级显示
- 当技能达到Lv.3时，技能卡片变灰并标记"已满级"
- 满级技能不出现在3选1中
