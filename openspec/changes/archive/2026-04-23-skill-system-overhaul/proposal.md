# 技能系统重构提案

**变更名称**: skill-system-overhaul  
**创建日期**: 2026-04-23  
**状态**: draft

---

## 一、变更目的

重构波次结束后的技能选择系统，将现有的7个技能统一为可叠加的增强系统，所有技能均支持分层强化，每级提供线性收益。

---

## 二、技能定义（全部可叠加，最大3级）

### 1. 穿透弹 (Pierce)
| 等级 | 效果 |
|------|------|
| Lv.0 | 无穿透，子弹击中1个敌人后消失 |
| Lv.1 | 穿透1个敌人，击中后继续飞行直到命中下一个 |
| Lv.2 | 穿透2个敌人 |
| Lv.3 | 穿透3个敌人 |

**叠加规则**: 每级+1穿透目标，最大3级

---

### 2. 散弹 (Scatter)
| 等级 | 效果 |
|------|------|
| Lv.0 | 无散弹，只有主子弹 |
| Lv.1 | +1路散弹（共2路） |
| Lv.2 | +2路散弹（共3路） |
| Lv.3 | +3路散弹（共4路） |

**叠加规则**: 每级+1路，最大3级

---

### 3. 蓄力弹 (Charge)
| 等级 | 效果 |
|------|------|
| Lv.0 | 无蓄力弹 |
| Lv.1 | 每隔一段时间自动发射1个充能炮弹（与普通子弹独立） |
| Lv.2 | 充能炮弹伤害+50% |
| Lv.3 | 充能炮弹发射频率+50% |

**叠加规则**: Lv.1解锁后Lv.2/Lv.3强化效果，最大3级

---

### 4. 光刀 (Blade)
| 等级 | 效果 |
|------|------|
| Lv.0 | 无光刀 |
| Lv.1 | 获得旋转光刀，可销毁来袭子弹 |
| Lv.2 | 光刀覆盖角度+45° |
| Lv.3 | 光刀覆盖角度+90° |

**叠加规则**: Lv.1解锁后增加覆盖范围，最大3级

---

### 5. 激光连锁 (Laser Chain)
| 等级 | 效果 |
|------|------|
| Lv.0 | 无激光连锁 |
| Lv.1 | 激光可连锁2个额外目标 |
| Lv.2 | 激光可连锁3个额外目标 |
| Lv.3 | 激光可连锁4个额外目标 |

**叠加规则**: 每级+1连锁目标，最大3级

---

### 6. 电磁波 (EM Wave)
| 等级 | 效果 |
|------|------|
| Lv.0 | 满能量发射1次电磁波 |
| Lv.1 | 发射2次波纹，余波伤害20% |
| Lv.2 | 发射3次波纹 |
| Lv.3 | 发射4次波纹 |

**叠加规则**: 每级+1波纹次数，最大3级

---

### 7. 充能加速 (Energy Charge)
| 等级 | 效果 |
|------|------|
| Lv.0 | 基础充能（每10伤害+10能量） |
| Lv.1 | 每10伤害+15能量 |
| Lv.2 | 每10伤害+20能量 |
| Lv.3 | 每10伤害+25能量 |

**叠加规则**: 每级+5能量/10伤害，最大3级

---

## 三、UI/交互改动

### 波次完成界面
- 显示当前拥有的所有技能等级（图标+等级）
- 3选1卡片展示3个随机技能选项
- 选项不再包含已满级技能
- 同一技能可重复出现以供叠加

### 技能卡片显示
```
[技能图标]
技能名称
效果描述
当前等级: X/3
```

## 五、验收标准

1. ✅ 所有7个技能均支持叠加，最多3级
2. ✅ 穿透弹：每级穿透+1个敌人
3. ✅ 散弹：每级+1路子弹
4. ✅ 蓄力弹：Lv.1解锁，Lv.2伤害+50%，Lv.3频率+50%
5. ✅ 光刀：Lv.1解锁，Lv.2角度+45°，Lv.3角度+90°
6. ✅ 激光连锁：每级+1连锁目标
7. ✅ 电磁波：每级+1波纹，余波20%伤害
8. ✅ 充能加速：每级+5能量/10伤害
9. ✅ 波次完成界面正确显示所有技能等级
10. ✅ 满级技能不在3选1中出现

## 六、技术改动

- `skillStacks` 最大值从4改为3
- 各技能效果计算函数按新数值调整
- UI渲染逻辑更新以显示当前等级
