**版本**：v3.0（Roguelike）

**状态**：规划中

## 一、游戏概述

| 项目 | 内容 |
|------|------|
| **游戏名称** | 几何风暴 GeoStorm（Roguelike模式） |
| **类型** | 俯视角弹幕射击 + Roguelike |
| **平台** | 网页（单HTML文件 + 云端后端） |
| **目标用户** | 休闲玩家 / Roguelike爱好者 |

## 二、核心定位

### 与原版区别

| 维度 | 原版 v2.x | Roguelike v3.0 |
|------|----------|----------------|
| 波次 | 5波+1Boss，时间驱动 | **10波次，每波清完才进入下一波** |
| 成长 | 无 | **局内+局外双成长系统** |
| 死亡惩罚 | 记录分数 | **局内永久死亡，从头开始** |
| 重玩价值 | 中 | **高（每次局内build不同）** |
| 难度 | 固定 | **可累积变强，降低通关难度** |

### 设计原则

1. **不逼氪**：局外用几何碎片肝，无需抽卡
2. **局内策略性**：波次间技能选择是核心决策点
3. **局内掉落+波次选择双系统**：比起单纯掉落更有深度
4. **几何主题强化**：所有道具、技能贴合几何主题

## 三、版本历史

### v2.x → v3.0 主要更新（2026-04-21）
- **Roguelike波次系统**：10个波次，每波消灭100%敌人才进入下一波
- **精英怪掉落**：随机刷新精英怪，掉落增益道具
- **波次间技能选择**：每波结束后从3张技能卡选择1张
- **武器库系统**：5种武器，局外解锁，局内可选
- **战机升级系统**：局外消耗几何碎片永久升级属性
- **几何碎片**：局外货币，通过通关获得

## 四、局内成长系统

### 4.1 波次系统

**核心规则**：
- 游戏共10个波次
- 每波必须消灭**100%敌人**才进入下一波
- 第5波、第10波为**Boss波次**
- 消灭80%敌人后出现**快速通关提示**（可选立即进入下一波，但奖励减少）

**波次设计**：

| 波次 | 敌人构成 | Boss |
|------|---------|------|
| 第1波 | 8个圆形 | 无 |
| 第2波 | 6个圆形 + 3个方形 | 无 |
| 第3波 | 5个圆形 + 4个方形 + 5个追踪者 | 无 |
| 第4波 | 4个圆形 + 5个方形 + 8个追踪者 + 2个六边形 | 无 |
| **第5波** | 3个方形 + 6个追踪者 + 4个六边形 | **第1个Boss** |
| 第6波 | 6个方形 + 10个追踪者 + 5个六边形 | 无 |
| 第7波 | 5个方形 + 12个追踪者 + 6个六边形 + 2个精英 | 无 |
| 第8波 | 8个追踪者 + 8个六边形 + 4个精英 | 无 |
| 第9波 | 10个六边形 + 6个精英 | 无 |
| **第10波** | 10个六边形 + 8个精英 | **第2个Boss（加强版）** |

**精英怪机制**：
- 第3波起随机出现精英怪（外观更大+发光+特殊颜色）
- 精英怪有额外掉落（1-2个增益道具）
- 精英怪击杀计入波次完成进度

### 4.2 精英怪掉落

**掉落类型**：

| 道具 | 效果 | 稀有度 |
|------|------|-------|
| 🔷 散弹碎片 | 子弹分裂成3发扇形（持续30秒） | 普通 |
| 🔷 穿透碎片 | 子弹穿透所有敌人（持续20秒） | 普通 |
| 🛡️ 护盾 | 抵挡下一次伤害 | 少见 |
| ⚡ 能量球 | +50能量 | 常见 |
| ❤️ 生命恢复 | 恢复1条生命 | 少见 |
| 💎 几何碎片 | +10局外货币 | 常见 |

**掉落规则**：
- 普通精英：掉落1个道具
- 强化精英（第7波起）：掉落2个道具

### 4.3 波次间技能选择

每波结束后，出现**技能选择界面**（暂停状态），从3张技能卡中选择1张：

#### 技能卡分类

**武器强化类**：

| 技能 | 效果 | 叠加 |

| 火力提升 | 伤害+25% | 可叠加3次 |
| 急速射击 | 射速+20% | 可叠加3次 |
| 弹速强化 | 子弹速度+15% | 可叠加2次 |
| 集束弹幕 | 子弹体积+30% | 可叠加2次 |

**能量强化类**：

| 技能 | 效果 | 叠加 |
|------|------|-----|
| 充能加速 | 击杀充能+50% | 可叠加2次 |
| 初始充能 | 初始能量+50 | 不可叠加 |
| 能量涌动 | 能量上限+50 | 可叠加2次 |

**生存强化类**：

| 技能 | 效果 | 叠加 |
|------|------|-----|
| 生命强化 | 初始生命+1 | 可叠加2次 |
| 铁壁 | 无敌时间+50% | 可叠加2次 |
| 疾风步 | 移动速度+20% | 可叠加2次 |

**技能强化类**：

| 技能 | 效果 | 叠加 |
|------|------|-----|

| 水波强化 | K技能范围+50% | 可叠加1次 |

**特殊弹药类**：

## 七、技术方案

### 7.1 数据结构

**局内数据（每局重置）**：
```javascript
{
  currentWave: 1,
  playerHP: 3,
  playerEnergy: 0,
  currentWeapon: 'basic',
  skills: [], // 已选技能列表
  activeBuffs: [], // 当前激活的buff
  waveEnemiesKilled: 0,
  waveTotalEnemies: 0
}
```

**局外数据（永久保存）**：
```javascript
{
  geoFragments: 0, // 几何碎片
  upgrades: {
    armor: 0,    // 护甲等级
    engine: 0,   // 引擎等级
    core: 0,     // 能量核心等级
    ammo: 0,     // 弹药专家等级
    fireRate: 0  // 射速专家等级
  },
  unlockedWeapons: ['basic'], // 已解锁武器
  bestWave: 0, // 最佳波次
  totalGamesPlayed: 0,
  totalWins: 0
}
```

### 7.2 存储方案

| 数据 | 存储方式 |
|------|---------|
| 局外数据 | localStorage + 云端同步（登录用户） |
| 局内数据 | 内存（每局重置） |
| 排行榜 | 云端SQLite |

### 7.3 API扩展

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/geostorm/progress` | GET/PUT | 获取/更新局外数据（需登录） |
| `/api/geostorm/sync` | POST | 同步本地数据到云端 |
